## 📌 Descripción

Diseño de una **gramática BNF** para un lenguaje imperativo que modela funciones, expresiones, estructuras de control y manejo de arreglos, considerando aspectos léxicos y sintácticos.

---

## ⚙️ Alcance del Lenguaje

El lenguaje desarrollado incluye:

* Funciones con retorno tipado y parámetros
* Tipos de datos: `int`, `float`, `bool`, `char`, `string`
* Expresiones aritméticas, lógicas y relacionales combinadas
* Arreglos unidimensionales y bidimensionales
* Entrada y salida estándar (`cin`, `cout`)
* Estructuras de control (`if`, `do-while`, `switch`)
* Comentarios de línea y bloque
* Función principal `__main__`
